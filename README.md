Foodie – Food Delivery Website (Capstone Project)
Theme
Food Delivery App
(Chosen from the Capstone Project themes – Page 1 of Assignment PDF)

Description
This project is a fully responsive Food Delivery landing page made using
HTML5, CSS3, Flexbox, CSS Grid, and Media Queries, following
all requirements from Lab Assignment 5.

Sections Implemented
✔ Header + Navigation
✔ Hero/Banner
✔ Categories (Grid)
✔ Featured Restaurants (Grid)
✔ Testimonials
✔ Contact Form
✔ Footer

Breakpoints Used
Desktop: ≥ 1024px
Tablet: ~768px
Mobile: ≤ 480px
Techniques Used
Semantic HTML (header, section, article, footer)
CSS Grid for cards
Flexbox for alignment
Responsive media queries
Google Fonts
Hover effects & shadows
How to View
1.Download the folder
2.Open index.html in any browser
3.Resize the browser to see responsiveness
